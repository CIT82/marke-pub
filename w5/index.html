What I built

I decided to use several different Bootstrap components to create a simple music album page. But the components that I used which are unique are the progress bar and the placeholder component. The progress bar is used to show the progress of the page loading. The placeholder component is used to show a loading state for the album information. This is useful when the album information is being fetched from a server and the user needs to wait for the information to be displayed. The placeholder component is a simple way to show the user that the information is being loaded.
